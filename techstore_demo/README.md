# TechStore - Open Source Demo Website
Website bán thiết bị công nghệ (bản demo) dùng để thử nghiệm AI như Augment / Cursor.

## Tính năng
- Trang chủ đơn giản
- Danh sách sản phẩm mẫu
- Giao diện responsive cơ bản
- Dùng HTML/CSS thuần

## Cách sử dụng
Upload toàn bộ mã nguồn này lên GitHub để tạo dự án open‑source.
